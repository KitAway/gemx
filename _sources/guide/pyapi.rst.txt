API definitions
====================

**gemx.py**

:doc:`gemx.py <python_api_detail/gemx_api>` describes all the functions you could use to work with the GEMX library realized on an FPGA board.
The functions are all written in C++ and python/gemx.py provide a wrapper to use them in Python.
